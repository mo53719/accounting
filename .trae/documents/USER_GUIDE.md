# 家庭记账本 · 用户操作说明

> 给最终用户的操作手册。**5 分钟上手**全部功能。

---

## 1. 首次使用

1. 安装 APP → 打开
2. 看到「**欢迎使用咪汪记账**」登录页
3. 首次使用 → 设置自定义账号 + 6 位以上密码 → 完成
4. 进入「**日常账本**」（默认账本）

> 密码仅存在本机，不会上传任何服务器。**请牢记！**忘记需重置账号（数据保留）。

---

## 2. 记一笔

1. 任意页面 → 底部右下角「**+**」按钮
2. 选择「**支出**」或「**收入**」
3. 选分类（如餐饮 / 工资）
4. 输入金额
5. （可选）选日期、支付方式、添加备注、上传图片
6. 点击「**保存**」

> 保存后立即同步到首页、账单页、统计页。

---

## 3. 查看账单

- **首页**：看今日/本月预算 + 收支概况 + 柱状图
- **账单页**（底部 Tab）：按日分组的所有流水
- **统计页**（底部 Tab）：柱状图/折线图/饼图/分类排行

---

## 4. 多账本

适合：家庭共用 / 个人 + 公司 / 旅行专项

1. 底部 Tab **我的** → **账本管理**
2. 点「**+ 添加账本**」→ 输入名称 + 选图标
3. 切换：首页顶部「**当前账本**」卡片 → 选账本

> 不同账本数据**完全隔离**，互不干扰。

---

## 5. 预算管理

1. 底部 Tab **我的** → **预算中心**
2. 选维度：日 / 月 / 年 / 全部 / 自定义
3. 输入金额 → 保存
4. 首页预算环实时显示剩余 / 超额

---

## 6. 主题切换

1. 底部 Tab **我的** → **主题**
2. 选「**标准纯净简约**」或「**海贼王人物包**」
3. 立即生效，重启 APP 自动保留

详见 [THEME_GUIDE.md](./THEME_GUIDE.md)。

---

## 7. 数据备份

**强烈建议每周备份一次！** 数据仅存本机，丢失/换机/重装后无法找回。

1. 底部 Tab **我的** → **数据备份**
2. **导出**完整 JSON → 传到电脑/网盘
3. **恢复**：选 JSON 文件 → 二次确认 → 完成

详见 [BACKUP_GUIDE.md](./BACKUP_GUIDE.md)。

## 10. 常见问题

### Q1：忘记密码怎么办？

底部「**忘记密码？重置账号密码**」→ 确认重置 → 重新设置。
**当前账本数据不会丢失**，但需重新登录。

### Q2：数据存在哪？

全部存本机 IndexedDB，**不上传服务器**。换机/重装需用备份恢复。

### Q3：怎么彻底删除 APP 数据？

设置 → 应用 → **家庭记账本** → 存储 → **清除数据**（⚠️ 不可恢复，请先备份）。

### Q4：能多设备同步吗？

当前**不支持云同步**。建议：
- 设备 A 导出 → 设备 B 导入（账本 ID 会变）
- 或一台设备使用

### Q5：账本能删除吗？

账本管理 → 长按账本 → 删除（**至少保留 1 个**）。

### Q6：怎么找回某笔账单？

底部 Tab **账单页** → 左/右切月份；
或顶部 **搜索框** → 输入关键词。

## 11. 数据安全承诺

- ✅ 数据**仅本机存储**，不上传任何服务器
- ✅ 密码本地哈希（不存明文）
- ✅ WebView 持久化权限已开启（杀进程数据不丢）
- ✅ 50MB 大文件保护（防设备异常）
- ✅ 备份文件含版本校验（防格式错乱）
