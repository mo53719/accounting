# UI 活力潮流风全面升级计划

## Why
用户认为当前界面难看，存在四大问题：布局排版单调、配色不够高级、emoji图标幼稚、缺少动画。用户偏好活力潮流风（大胆配色、趣味插画、活泼动画），并要求密码输入页不要占太大位置。

## Current State Analysis
- 所有页面使用统一的 `card-cute` 白色卡片，缺乏层次感和视觉差异
- 配色以粉色 #FF8A9E 为主，过于单一，缺少对比和活力
- 图标全部使用 emoji，在不同平台显示不一致，显得幼稚
- 几乎没有动画效果，页面切换生硬
- EntryView 密码输入区域占据整个屏幕，功能亮点卡片占位太多
- 底部导航栏样式基础，缺少活力

## Proposed Changes

### 1. 全局样式升级 (`src/assets/styles/main.css`)
- **配色方案**：从单一粉色升级为多色活力配色
  - 主色：珊瑚橙 #FF6B6B → 渐变 #FF6B6B ~ #FF8E53
  - 辅色：薄荷绿 #4ECDC4、阳光黄 #FFE66D、梦幻紫 #A855F7
  - 背景：从纯色 #FFF9FA → 多彩渐变光晕 + 微妙纹理
- **新增 CSS 动画**：
  - `slide-up` 页面进入动画
  - `pulse-glow` 按钮脉冲发光
  - `wiggle` 图标抖动
  - `gradient-shift` 渐变流动
- **卡片升级**：新增 `.card-vivid`（渐变边框卡片）、`.card-glass`（毛玻璃卡片）
- **按钮升级**：新增 `.btn-vivid`（渐变按钮带 hover 发光效果）
- **插画装饰**：新增 `.deco-blob`（彩色色块装饰）、`.deco-dots`（圆点装饰）

### 2. 登录页精简 (`src/views/EntryView.vue`)
- **缩小密码区域**：数字键盘按键从 h-12 缩小到 h-10，间距收紧
- **移除功能亮点卡片**：3个功能卡片太占空间，改为顶部一行小标签
- **添加插画装饰**：顶部放一个大的 SVG 插画（钱袋子/小猪存钱罐风格），替代 bounce 的 emoji
- **背景**：改为多彩渐变 + 浮动色块装饰
- **动画**：密码点输入时有弹跳动画，确认时有成功动画

### 3. 首页升级 (`src/views/HomeView.vue`)
- **顶部区域**：渐变从单色粉 → 珊瑚橙到暖黄的多色渐变
- **汇总卡片**：从半透明白 → 毛玻璃效果 + 渐变边框
- **今日支出**：改为独立的小卡片，带脉冲动画
- **最近记录**：每条记录增加滑入动画（staggered）
- **空状态**：用插画 SVG 替代 emoji
- **波浪分隔线**：改为更流畅的多层波浪 + 渐变色

### 4. 记账页升级 (`src/views/AddView.vue`)
- **顶部**：渐变背景 + 插画装饰
- **金额输入**：大字体 + 渐变下划线 + 输入时数字弹跳动画
- **分类选择**：每个分类改为圆形图标容器（带渐变背景环），选中时有弹跳 + 发光效果
- **支付方式**：改为带图标的圆角胶囊按钮，选中时有渐变效果
- **记账人**：改为头像风格圆形按钮
- **保存按钮**：渐变色 + 脉冲发光动画

### 5. 统计页升级 (`src/views/StatsView.vue`)
- **顶部**：渐变背景 + 插画装饰
- **收支概览**：改为渐变背景卡片，带图标动画
- **图表切换**：按钮改为胶囊标签样式，带滑动指示器
- **饼图区域**：外围加装饰色块
- **趋势图区域**：加渐变填充

### 6. 账单列表页升级 (`src/views/ListView.vue`)
- **添加日期分组**：按日期分组显示，每组有日期标题
- **日期标题**：渐变背景小标签
- **记录项**：增加滑入动画
- **空状态**：插画风格

### 7. 设置页升级 (`src/views/SettingsView.vue`)
- **顶部**：渐变背景
- **卡片**：使用 `.card-vivid` 渐变边框
- **记账人列表**：头像风格圆形图标 + 渐变背景

### 8. 底部导航升级 (`src/components/BottomNav.vue`)
- **中间添加按钮**：改为更大的渐变圆形，带脉冲发光动画
- **图标**：选中时放大 + 弹跳动画
- **背景**：毛玻璃效果
- **指示器**：选中项下方加渐变小条

### 9. 交易项组件升级 (`src/components/TransactionItem.vue`)
- **分类图标**：从方形圆角 → 圆形容器 + 渐变背景环
- **金额**：加粗 + 渐变色（收入绿渐变、支出红渐变）
- **记账人标签**：改为小头像风格
- **整体**：左侧加一条彩色竖线指示分类颜色

### 10. SVG 插画组件 (`src/components/Illustrations.vue`)
- 创建几个简单的 SVG 插画组件：
  - `PiggyBank` - 小猪存钱罐（登录页）
  - `EmptyState` - 空状态插画（首页/列表）
  - `ChartDecor` - 图表装饰（统计页）

## Assumptions & Decisions
- 使用纯 CSS + SVG 实现插画效果，不引入额外图片资源
- 动画使用 CSS animation + Vue transition，不引入额外动画库
- 保持 TailwindCSS 为主，自定义 CSS 为辅
- 图标仍使用 emoji 但用更好的容器样式包裹，不引入图标库（保持轻量）
- 渐变和动画注意性能，避免影响低端手机

## Verification Steps
1. `npx vite build` 编译通过
2. 浏览器打开 http://localhost:5173/ 检查每个页面视觉效果
3. 检查动画流畅度（无卡顿）
4. 检查手机端适配（max-width: 480px）
5. 检查所有功能正常（记账、删除、切换月份等）
